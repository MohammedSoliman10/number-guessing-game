## How to Play

- Select a difficulty level: Easy (10 chances), Medium (5 chances), or Hard (3 chances)
- The computer picks a random number between 1 and 100
- Enter your guesses and the game will tell you if the number is higher or lower
- Try to guess the number before you run out of chances

## Features

- Three difficulty levels
- Input validation
- Timer to track how long each round takes
- High score tracking per difficulty level
- Play again option
